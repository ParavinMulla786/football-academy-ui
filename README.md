## 📌 Project Overview

**Football Academy UI** is a modern, clean, and responsive frontend website built using **HTML, CSS, and Bootstrap**.  
This project is developed by converting a **Figma design into a real website**, focusing on UI/UX, layout structure, and responsiveness.

It represents a professional football academy website with visually appealing sections and smooth user experience.
